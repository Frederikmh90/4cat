Videos are scraped in the order they are returned by BitChute's search function.

You can scrape results for up to fifteen items at a time. Separate the items with commas or blank lines. When searching for usernames, there is no need to include @ in front.

You can optionally scrape more details - exact publication date, likes, dislikes, category, comment count and channel subscriber count - for each video. Note that this takes a couple of seconds per video (which can add up!). Consider doing a basic query first and then repeating it with more details only if necessary.
