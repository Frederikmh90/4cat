Douban is a Chinese internet forum hosting a variety of thematic groups in which people can post threads and comment on them. This data source allows you to scrape recent threads for a given list of groups.

Note that Douban severely limits the retrieval of older content. Therefore this data source can only scrape the most recent topics in a given group. You can optionally limit the scraped topics to a given date range, but note that typically only the 500 or so most recent topics in a group will be available for scraping.

You can enter the groups to scrape as group IDs (e.g. 687802) or URLs (e.g. https://www.douban.com/group/687802/. Separate multiple groups with commas or new lines. If you enter more than 25 groups, only the first 25 will be scraped.
